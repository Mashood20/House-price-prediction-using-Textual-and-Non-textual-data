# Experiment Environment Setup

1. Install Anaconda 

2. Create and Activate the Environment:
   - Open a terminal and navigate to the directory containing `Experiments_env.yaml`.
   - Create the environment by running the command:
     ```
     conda env create -f Experiments_env.yaml
     ```
   - After creation, activate the environment:
     ```
     conda activate Experiment_env
     ```

## Running Experiments
- Once the environment is activated, you can run the experiments.
